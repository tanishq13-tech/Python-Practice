# Python Practice

A curated collection of Python exercises and examples covering the fundamentals of the language.

This repository is organized by topic and includes hands-on practice files for beginners learning Python basics, data structures, control flow, functions, object-oriented programming, file handling, and GUI programming.

## Contents

- `4 June (Introduction To Python)` - Python basics and introductory examples
- `5 June (Operators)` - Arithmetic, assignment, and comparison operator exercises
- `6 June (Conditional Statements)` - `if`, `if-else`, nested conditions, and related problems
- `8 June (Loops)` - `for` and `while` loop practice, series, tables, and pattern tasks
- `9 June (Problems)` - Classic programming problems like palindrome, Armstrong, and prime checks
- `11 June (List & Tuple)` - List and tuple operations, common element checks, and aggregation tasks
- `12 June (Set)` - Set operations, min/max, and sum exercises
- `15 June (Dictionary)` - Dictionary creation, traversal, sorting, and utility operations
- `16 June (Functions)` - Function examples and reusable code patterns
- `18 June (Recursion)` - Recursive solutions for common problems
- `19 June (OOP's)` - Object-oriented programming examples and class-based designs
- `22 June (File & Exception Handling)` - Reading, writing, and exception-safe file handling
- `23 June (Library_Tkinter)` - Basic GUI examples using Tkinter layout managers

## How to use

1. Open the repository folder in your code editor.
2. Navigate to a topic folder.
3. Open a Python file (`.py`) to inspect the example or exercise.
4. Run the file using Python from the terminal:

```powershell
python <filename>.py
```

## Recommendation

- Start with the introductory files in `4 June (Introduction To Python)`.
- Move through operators, conditionals, loops, lists, sets, and dictionaries.
- Finish with file handling and Tkinter to see practical applications.

## Notes

- Files are named with a lesson date and a short description for easy browsing.
